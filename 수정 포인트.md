- tier = "협력사 차수"(원청 기준 거리)가 본래 의미. 부품 라벨 아님.

- 따라서 parts.tier_level 은 0-base:
    0 Pack(원청) / 1 Module / 2 Cell / 3 활물질(CAM·ANO) / 4 전구체·정제리튬 / 5 제련 / 6 광산

  같은 회사가 여러 단계를 겸업하면(예: 한양셀=Module+Cell) hop으로 분리해 연속 hop에 중복 등장.
